About this application:

Few features for the front end of this website, all need to be complete this sprint:

  1. Create search bar
  2. Create a spot to upload files (drag and drop style>
  3. If necessary, create a nav bar for all endpoints

Goals for this sprint:

  • A search bar that validates user input, and breaks down search into relevant keywords (all in the frontend)
  • An upload function that allows user to upload multiple files of only type .cpp
  • Create error and success cards that are displayed to user, notifying any updates from the webapp

Format of this application:

  • There are seperate folders for any css and js files you would like to add. If you would like to add files, make sure to add them to the index.html as well
  • The majority of formatting and styling is done by bootstrap (currently pointing to a cdn). The primary reason is for ease of responsive and clean design
